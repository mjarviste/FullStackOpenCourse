Here is a simple flow chart:

```mermaid
sequenceDiagram
    participant User
    participant Browser
    participant Server

    User->>Browser: Write text and click Submit button
    Browser->>Server: HTTP POST request to /new_note
    Server->>Browser: Respond with HTTP GET request to /notes
    Browser->>Server: HTTP GET request to /notes
    Server->>Browser: Serve Notes page
    Browser->>Server: HTTP request to fetch the style sheet
    Server->>Browser: Serve style sheet
    Browser->>Server: HTTP request to fetch the JavaScript code
    Server->>Browser: Serve JavaScript code
    Browser->>Server: HTTP request to fetch raw data from JSON file
    Server->>Browser: Serve raw data from JSON file
```
